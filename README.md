### 快捷键
>command/control+"+"    放大
>
>command/control+"-"    缩小
>
>command/control+U      全局查看与窗口查看切换
>
>command/control+z      撤销
>
>command/control+shift+z    回退
>
>command/control+D+触摸板    画板拖动与缩放

### run example

```
yarn  install

npm run  build:all

npm run start:example
```
